Catnip

Chairy Madness

My second game, written in C++, again using SFML.

The Assets except for the font were created by me, the code is by me and a colleague of mine.

This game is a local multiplayer game where two players compete against eachother by stacking chairs. Players take turns swinging and throwing chairs, trying to create a tower and always stacking higher than their competitor.

Pan-Tastic Wizard Adventures

This game is made with Unity, so the Scripts are written in C#.

The Assets except for the font and the environment were created by me, the code is by me and two colleagues.

In this 2D platformer, a player is a small wizard who casts spells by combining collected ingredients in a pan. Different ingredients have different effects. Spell effects are determined by the position the ingredient is in. The effects are damage type, flight trajectory and number of projectiles that can be shot when cast one time.
